# mquiz
### Product
A REST API for creating a quiz or attempting an existing one.
### Project
A microservices backend with managed service registery and load balancing.
### Tech stac
Spring Boot, Spring Data JPA, MariaDB, Netflix Eureka, and OpenFeign.
